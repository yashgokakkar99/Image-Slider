# Image-Slider
Using ReactJs and tailwindcss

Step1 : npx create-react-app img-slider

Step2 : cd img-slider

Step3 : npm install react-icons --save

Step4 : npm install -D tailwindcss
        npx tailwindcss init
        
Step5 : In tailwind.config.js

        Change :  content: [
    "./src/**/*.{js,jsx,ts,tsx}",
  ],
  
Step6 : In index.css
        @tailwind base;
        @tailwind components;
        @tailwind utilities;
